# Gym Management System (C++)

A simple console-based Gym Management System built using C++ to manage trainees, trainers, and classes.

## Features
- Admin: Add/view trainers and classes  
- Trainer: Login system  
- Trainee: Register, login, view classes  
- File handling using `.txt` files  

## Requirements
- C++ Compiler (g++)  
- Visual Studio Code (optional)  

## How to Run
```bash
git clone https://github.com/ayuushpokharel/gms
cd gms
g++ main.cpp -o gms
./gms
````
